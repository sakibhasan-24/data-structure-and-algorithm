## 🛠️ Phase 1: The Foundation (Month 1)
*Focus: Rebuilding algorithmic intuition and mastering linear patterns.*

### **Week 1-2: Arrays, Strings & Two Pointers**
- [ ] **Core Patterns:** Two Pointers, Sliding Window, Prefix Sum.
- [ ] **Must-Solve (C++):** - [ ] Pair with Target Sum (Two Pointers)
    - [ ] 3Sum (Sorting + Two Pointers)
    - [ ] Longest Substring Without Repeating Characters (Sliding Window)
- [ ] **Daily Task:** 1 Easy, 1 Medium LeetCode.

### **Week 3-4: Linked Lists & Hashing**
- [ ] **Core Patterns:** Fast & Slow Pointers, Reversing Lists, Hash Map frequency counting.
- [ ] **Must-Solve (C++):** - [ ] Linked List Cycle Detection
    - [ ] Merge Two Sorted Lists
    - [ ] Group Anagrams (Hashing)
- [ ] **Project:** Set up the base architecture for the **Next.js AI Image Generator**.

## 🌳 Phase 2: Nonlinear Thinking (Month 2)
*Focus: Mastery of Recursion, Trees, and Graphs.*

### **Week 1-2: Trees & Binary Search Trees (BST)**
- [ ] **Core Patterns:** DFS (Pre/In/Post order), BFS (Level Order), Tree Recursion.
- [ ] **Must-Solve:** - [ ] Maximum Depth of Binary Tree
    - [ ] Lowest Common Ancestor
    - [ ] Binary Tree Level Order Traversal
- [ ] **Note:** Focus on visualizing the call stack for recursive functions.

### **Week 3-4: Backtracking & Basic DP**
- [ ] **Core Patterns:** State-space search, Memoization (Top-down DP).
- [ ] **Must-Solve:** - [ ] Permutations / Subsets
    - [ ] Combination Sum
    - [ ] Climbing Stairs (Intro to DP)
- [ ] **Project:** Integrate **Zod & React Hook Form** into your AI Image App for robust validation.

*Focus: Dynamic Programming and High-Level System Design.*

## 📖 Study Tips for 2026
1. **The 3-Day Rule:** If you struggle with a problem today, re-solve it from scratch in 3 days.
2. **Quality over Quantity:** Understanding *why* a pattern works is better than memorizing 500 solutions.
3. **Bangla to English:** Practice explaining your C++ logic in English, as FAANG interviews will likely be in English.
